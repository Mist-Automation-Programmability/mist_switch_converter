Information

This site is trying to convert Cisco IOS configuration files to Juniper Mist Switch Templates.
All the configuration elements are not converted, only the most common ones.
Your configuration files are not sent to the server. All the processing is directly done inside your web browser.
If you device to automatically deploy the template to Mist from here, the template content will be first sent to a 3rd part server then to the Mist Cloud.

The purpose of this site is to convert Cisco IOS and Junos configuration files to Juniper Mist Switch Templates.
Your configuration files are not sent to the server. All the processing is directly done inside your web browser.
Converted parameters:
Common parameters
NTP Servers
DNS Server
DNS Domain
SYSLOG Servers
CLI Banner
Authentication
RADIUS Authentication and Accounting Servers
TACACS+ Servers
VLANs
VLAN IDs
VLAN Names (For Cisco, requires the VLAN Database)
DHCP Snooping
Interfaces
Mode (Access/Trunk)
VLANs (access, native/untaggued, trunk allowed, voice VLAN)
Speed/Duplex or Auto Negociation
Authentication (dot1x, MAC Auth, MAC Auth only, rejected/guest VLAN)
MAC Limit
Persitant MAC
MTU
Auto QoS
PoE (enabled/disabled)
Admin Status (enabled/disabled)
STP Edge
To get a good result, each Junos file must contain the configuration from a single switch with the result of the show | display inheritance | display set command.
The Juniper interface-ragnes

Information

The purpose of this site is to convert Cisco IOS configuration files to Juniper Mist Switch Templates.
Only the following components are converted:
  • NTP Servers
  • DNS Server
  • DNS Domain
  • RADIUS Authentication and Accounting Servers
  • TACACS+ Servers
  • SYSLOG Servers
  • VLANs
  • DHCP Snooping
  • Part of the port profiles based on the Cisco switch ports configuration
  • CLI Banner
Your configuration files are not sent to the server. All the processing is directly done inside your web browser.
